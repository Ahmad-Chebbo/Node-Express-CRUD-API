## Introduction

Sample REST API using Node JS and Express

## Developing

**Getting Started**

*Clone this repo:*

```
git clone https://github.com/Ahmad-Chebbo/Node-Express-CRUD-API

cd Node_Express_CRUD_API
```

*Install dependencies:*

```
npm install && npm start
```
